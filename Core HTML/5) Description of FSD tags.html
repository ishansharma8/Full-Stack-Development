Description list of Full stack development tech stack

HTML
HTML defines the structure of webpage
CSS
CSS is used to style the content of a web page by adding design elements like colors, fonts, and spacing
Javascript
JavaScript is used to add dynamic behavior to the webpage and add special effects to the webpage
React.js
React helps us to make website by using components.The components can be reused again
Node.js
Node.js is JavaScript runtime.It is used for building fast and scalable server-side applications.
Express.js
Express.js provides an easy way to create web server and render HTML pages for different HTTP requests by configuring routes for your application.
MongoDB
MongoDB uses documents that can contain sub-documents in complex hierarchies making it expressive and flexible
